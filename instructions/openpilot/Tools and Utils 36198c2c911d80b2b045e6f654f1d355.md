# Tools and Utils

This doc defines all tools and utils that are public for agents.

## File Reader

This function read files as context efficiently. 

Compared with traditional agent tools, we use different bounds for reading files. For example, for the database file, we read the first several lines as as an example. But for the code file, we read all of it to make sure the functions fully recognized.

Input: file path

Output: file content

## File Writer

This function write files.

Input: file path, content

Output: None (directly write files)

## Command Executor

This function execute given command in the system.

Input: command

Output: system output (error or correct running results)

## Embedder

This function utilizes an embedding model to embed the text into an embedding vector, which is usually used for extracting the semantics of the text.

Input: Query

Output: Embedding Vector
